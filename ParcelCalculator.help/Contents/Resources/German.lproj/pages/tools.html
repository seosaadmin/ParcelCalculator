Tools

Benutzen Sie die integrierten Tools für spezifische Aufgaben, erreichbar über das Hauptmenü. Die DPD Konformitätsprüfung verwendet das Gurtmaß eines Pakets um zu evaluieren, ob ein Versand über Direct Parcel Distribution möglich ist, oder eben nicht. Um dieses Tool erfolgreich auszuführen, müssen Sie die Höhe, Breite und Länge einer Sendung bereitstellen. Parcel Calculator wird bemängeln, wenn Sie die Konformitätsprüfung ohne brauchbare Werte starten.
