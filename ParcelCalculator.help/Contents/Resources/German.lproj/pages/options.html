Optionen

Parcel Calculator verwendet Sparkle für automatisierte Updates. Klicken Sie den 'Update' Button in der Symbolleiste oder verwenden Sie den entsprechenden Eintrag im Hauptmenü um zu prüfen ob eine neuere Version vorliegt. Das generelle Verhalten im Umgang mit Updates kann über die Einstellungen modifiziert werden. Sie können Parcel Calculator anweisen automatisch nach Aktualisierungen zu suchen, und sie können sogar festlegen in welchem Intervall dies zu geschehen hat (stündlich, täglich, wöchentlich, monatlich). Natürlich können Sie automatische Updates auch untersagen. Rufen Sie die Einstellungen mit dem Kürzel ⌘, oder dem Eintrag im Hauptmenü auf. Es sei anzumerken, dass sie über das Hilfsmenü weiterführende Links rund um Parcel Calculator erreichen können. Besuchen Sie die Homepage, das Source Repository oder melden Sie Fehlverhalten bzw. Bugs in der Anwendung.
