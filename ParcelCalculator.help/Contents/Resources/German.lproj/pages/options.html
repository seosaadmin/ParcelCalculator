Einstellungen

Tracking: Sicherlich möchten Sie festlegen, welcher Tracking Modus beim Programmstart angewandt werden soll. Hier haben Sie die Möglichkeit. Zusätzlich finden Sie eine Option zum Löschen aller Einträge in der Paketdatenbank, ein nicht umkehrbares Feature.

Maßheinheiten: Mit welchen Maßeinheiten möchten Sie berechnen? Zur Wahl stehen das metrische System mit Zentimetern und Kilogramm, sowie das US-customary System basierend auf Zoll und Pfund.

Updates: Parcel Calculator verwendet Sparkle für automatisierte Updates. Klicken Sie den 'Update' Button in der Symbolleiste oder verwenden Sie den entsprechenden Eintrag im Hauptmenü um zu prüfen ob eine neuere Version vorliegt. Das generelle Verhalten im Umgang mit Updates kann über die Einstellungen modifiziert werden. Sie können Parcel Calculator anweisen automatisch nach Aktualisierungen zu suchen, und sie können sogar festlegen in welchem Intervall dies zu geschehen hat (stündlich, täglich, wöchentlich, monatlich). Natürlich haben sie die Möglichkeit, automatisierte Updates zu untersagen.
