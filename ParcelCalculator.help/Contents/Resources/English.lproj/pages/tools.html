Tools

Parcel Calculator comes with a set of built-in tools for special tasks. DPD conformity test checks the combined dimensions of a parcel to find out wether a shipment with Direct Parcel Distribution is possible or not. To run this tool you will need to provide height, width and length of parcel. Parcel Caclulator complains if you try to run the DPD conformity test and one of those values is missing.
