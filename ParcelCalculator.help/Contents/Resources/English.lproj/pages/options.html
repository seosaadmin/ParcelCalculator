Preferences

Tracking: You may want to enable a particular tracking mode on startup, do it here. Additional there is an option for erasing all entries in your parcel library. Note that this feature is irreversible.

Measurements: This is the place to specify a measurement system. You can choose either the metric system, based on Centimeters and Kilogram or the US-customary system with Inches and Pounds.

Updates: Parcel Calculator uses Sparkle for automated updates. Hit the 'Update' button in the toolbar or the corresponding entry in the main menu to check for available updates. You can set up Parcel Calculator to search automatically for updates and you can even specify an interval for it (hourly, daily, weekly, monthly). Of course you are able to deactivate any automated updates.
